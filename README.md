# Svara

Svara is a Flutter-based Android music app with streaming, search, queue management, and offline playback support.

## Links

- GitHub: https://github.com/codewithevilxd/svara
- Developer: https://github.com/codewithevilxd
- Portfolio: https://nishantdev.space
- Email: codewithevilxd@gmail.com

## Backend

- Configured API: https://rf-snowy.vercel.app/

## Features

- Search songs, albums, artists, and playlists
- Stream music with background playback
- Download tracks and albums for offline listening
- Persistent queue, recent playback, and liked songs
- Multi-language discovery and cached home sections

## Setup

```bash
git clone https://github.com/codewithevilxd/svara.git
cd svara
flutter pub get
flutter run
```
